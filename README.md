Игра «Крестики-нолики» (Tkinter)

Описание проекта

Данный проект представляет собой настольную игру **«Крестики-нолики»**, реализованную на языке программирования **Python** с использованием стандартной библиотеки **Tkinter** для создания графического пользовательского интерфейса.
Игра предназначена для двух игроков, которые поочерёдно делают ходы, размещая символы **X** и **O** на игровом поле размером 3×3. Цель игры — первым выстроить три одинаковых символа по горизонтали, вертикали или диагонали.
Проект разработан в рамках **курсовой работы** и демонстрирует навыки работы с GUI, обработкой событий и логикой игры.

Используемые технологии:
* **Python 3**
* **Tkinter** (стандартная библиотека Python)
* ООП (функциональный и событийно-ориентированный подход)

Функциональные возможности:
* Графический интерфейс пользователя
* Игровое поле 3×3
* Поочерёдные ходы игроков (X / O)
* Автоматическое определение победителя
* Определение ничьей
* Блокировка поля после завершения игры
* Возможность начать новую игру

Интерфейс:
Интерфейс реализован с помощью кнопок Tkinter, каждая из которых соответствует клетке игрового поля. Управление осуществляется с помощью кликов мыши.

Логика игры:
* Проверка победы выполняется после каждого хода
* Победа определяется по всем возможным комбинациям:
  * строки
  * столбцы
  * диагонали
* При заполнении всех клеток без победителя фиксируется ничья

Цель курсовой работы:
* Закрепление навыков программирования на Python
* Изучение основ GUI-разработки
* Реализация событийной логики
* Разработка законченного пользовательского приложения
